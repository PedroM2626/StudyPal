# Project built by Curling AI

This project was built by curling AI from ground up

## Instalação

Este projeto utiliza pnpm como gerenciador de pacotes. Para instalar as dependências, execute:

```bash
pnpm install
```

## Scripts Disponíveis

- `pnpm dev` - Inicia o servidor de desenvolvimento
- `pnpm build` - Gera a build de produção
- `pnpm preview` - Visualiza a build de produção
- `pnpm test` - Executa os testes
- `pnpm lint` - Executa o linter
- `pnpm format` - Formata o código
